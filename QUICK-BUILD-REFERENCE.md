- âŒ Head + Hands both Bombardero (duplicate Weapons stat)
- âŒ Class Item same as any other (poor diversification)

### Stat Conflicts
- âŒ Grenade 200 + Melee 200 + Class 200 (impossible with weapons/health/super)
- âŒ All stats evenly distributed (weak focus, no viable playstyle)
- âŒ Super 0, Grenade 0 (build has no core engine)

### Mod Conflicts
- âŒ 3+ Barrier mods (stacking cap = 2 only)
- âŒ 2x Font of Might (energy duplication)
- âŒ 2x Recharge Overload (overcap ability generation)
- âŒ Font of Might + Font of Power (conflicting triggers)
- âŒ Well of Life + Melee Wellmaker + Grenade Wellmaker (well cap overflow)

### Aspect/Fragment Conflicts
- âŒ Solar aspect without Solar fragment options (limits synergy)
- âŒ 2 aspects that compete for same fragment slots (choose 1)
- âŒ Aspects from different subclasses (only 1 per subclass allowed)

### Ability Conflicts
- âŒ Melee-focused build with Empowering Rift (wastes ability uptime)
- âŒ Grenade-spam build with 0 Grenade stat (defeats purpose)
- âŒ Super-cycling build without weapon damage stats (dps window fails)
