# Build Templates

This directory contains example build templates for D2-Guardian-Forge.

## Available Templates

### grenade-spam-warlock.json
- **Class**: Warlock
- **Subclass**: Solar
- **Focus**: Maximum grenade uptime
- **Stats**: Grenade 190 (2 charges), Melee 150
- **Best For**: Strikes, Dungeons, Add Clear

## Usage

These templates can be imported into the build planner or used as reference for creating your own builds.

## Contributing

Feel free to add your own build templates! Follow the JSON schema defined in `/src/types/index.ts`.
